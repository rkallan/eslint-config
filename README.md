# eslint-config

> Shareable [ESLint](https://eslint.org/) config.

## Usage

**Install**:

```bash
npm install rkallan-eslint-config
```

**Usage in `.eslintrc.js`**:

```js
module.exports = {
    extends: ["rkallan"],
};
```

run `npx sort-package-json` to sort package.json
run `npx npm-check-updates` to check packages for updates

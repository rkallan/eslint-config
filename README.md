# eslint-config

> Shareable [ESLint](https://eslint.org/) config.

## Usage

**Install**:

```bash
npm install rkallan-eslint-config
```

**Usage in `.eslintrc.js`**:

```js
module.exports = {
    extends: ["rkallan-eslint-config"],
};
```

NOTE: On config error use ./node_modules/rkallan-eslint-config

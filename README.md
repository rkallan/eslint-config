# eslint-config

> My personal [Prettier](https://prettier.io) config.

## Usage

**Install**:

```bash
npm install rkallan-eslint-config
```

**Usage in `.eslintrc.js`**:

```js
module.exports = {
    extends: ["./node_modules/rkallan-eslint-config"],
};
```
